# Midnight Journal

A simple TUI Journal for [Midnight](https://midnight.hackclub.com)

## Features

- A simple TUI based interface
- Save you Journals
- Markdown rendering
- Auto scroll


## Setup(macOS)

> [!NOTE]
> Most of these commands can also be changed for windows, but I don't own one so I can't test it.

### Make a virtual environment and activate it

```
python3 -m venv .venv
source .venv/bin/activate
```

### Install dependencies

```
uv sync
```

### Run Project

```
uv run app.py
```

## Development

### Install `textual-dev`

```
uv add textual-dev
```
